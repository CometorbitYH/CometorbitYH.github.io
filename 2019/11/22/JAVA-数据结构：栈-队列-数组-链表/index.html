    数组（Array）

    数组是有序的元素队列，是在内存中开辟了一段连续的空间，在此段空间存放元素

    特点：长度固定，查询快，增删慢

    为什么增删慢？

    因为数组是一个固定的长度，增加和删除的话，都要重新创建一个新数组，并将旧数组中还保留的元素都复制到新数组，还要考虑删除或增加中间元素时，后面元素都要往前挪或往后挪，之后垃圾系统会回收旧数组，因此数组增删元素效率很低。

      • 释放节点资源
